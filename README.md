# 🚀 zapretGUI-discord-youtube-roblox

Удобный графический интерфейс для управления службой **Zapret**. Забудь про консоль и батники — запускай всё одной кнопкой!

> **Важно:** Запускайте программу от имени **Администратора**.

---

### 🛠 Как пользоваться:
1. Скачайте архив из раздела **Releases**.
2. Распакуйте в любую папку.
3. Запустите `KiloFarsh_Zapret.exe`.
4. Введите нужную стратегию (или оставьте стандартную) и нажмите **СТАРТ**.
